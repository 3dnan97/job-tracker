# Job Application Tracker

A polished React + TypeScript application for tracking job applications efficiently.

## Core Features
- **Job Management:** Track application status, company details, roles, and interview progress.
- **Privacy & Security:** Per-user data isolation via Supabase and protected route guards.
- **Modern UI:** Built with Tailwind CSS for a responsive, clean experience.
- **Architecture:** Scalable routing with React Router and state management optimized for performance.

## Tech Stack
- **Frontend:** React, TypeScript, Vite
- **Routing:** React Router
- **Backend/Auth:** Supabase
- **Styling:** Tailwind CSS

## Project Structure
- `src/components`: Reusable UI components (Layout, Nav, etc.)
- `src/pages`: Route-level screens
- `src/types`: Shared TypeScript interfaces
- `src/i18n`: Centralized UI strings

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
